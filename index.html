Contribute.md

Contribute.md is an initiative to get more awesome people involved in open source projects. It’s a tiny markdown file that lives inside your project repository and welcomes people to take part.

What is Contribute.md

All help is welcome
For many beginners the world of open source is complicated and scary. However making new features isn’t the only way you can contribute to an open source project. Contribute.md makes it easier to find ways to help

Simple and straightforward
Contribute.md is a way to give everyone the tools to help in their favorite open source project, whether they are technical or not. It makes contributing friendly, simple and straightforward by providing a template maintainers can fill to help new community members to contribute.

Open source is for everyone
Open source is important because XYZ. However, maintaining a project can be time consuming and all types of help are welcome. Including people with different skillsets to your project will make it even more awesome.
